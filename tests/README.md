# ForceQUIT Tests

## Test Structure

This directory contains tests for the ForceQUIT application.

### Current Status
- **Unit Tests**: To be implemented for core functionality
- **Integration Tests**: To be implemented for UI components  
- **Performance Tests**: To be implemented for force quit operations

### Test Framework
- **XCTest**: Primary testing framework for Swift
- **SwiftUI Testing**: For UI component testing

### Running Tests
```bash
swift test
```

### Test Coverage Goal
Target: >90% code coverage for core functionality

## Test Categories

### Unit Tests
- Application lifecycle
- Process enumeration
- Force quit operations
- Error handling

### UI Tests  
- Interface responsiveness
- User interactions
- Visual feedback

### Performance Tests
- App launch time
- Force quit speed
- Memory usage
