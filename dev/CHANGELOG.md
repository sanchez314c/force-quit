# Changelog

All notable changes to ForceQUIT will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Project standardization with universal folder structure
- Comprehensive build system with multi-platform support
- Enhanced documentation structure

### Changed
- Reorganized project structure for better maintainability
- Consolidated build scripts into organized directories

## [2.0.0] - 2025-08-27

### Added
- SWARM 2.0 AI development framework integration
- Multi-modal activation system (GUI, menu bar, hotkeys, shake detection)
- Advanced security framework with SIP compliance
- Performance monitoring and analytics system
- Universal binary support (Intel + Apple Silicon)
- Comprehensive test suite with security and performance tests
- Auto-updater system
- Build pipeline with code signing and notarization

### Changed
- Complete architectural redesign using MVVM pattern
- Enhanced UI with SwiftUI and custom animations
- Improved security model with sandboxing and helper tools
- Modular package structure with separate frameworks

### Security
- SIP compliance validation
- Secure privilege escalation with helper tools
- Sandbox security manager implementation
- Authorization framework with proper entitlements

## [1.0.0] - Initial Release

### Added
- Basic force quit functionality
- Simple macOS application interface
- Core process management features
